
# go-date-example


## Installation

```shell
# 1. clone repository
git clone https://github.com/bagus123/go-date-example.git

# 2. downloads all dependencies and build binary
go build

# run from binary
./go-date-example

# or run from source
go run main.go

# note
# clean cache go
go clean --modcache

# remove unused module
go mod tidy
```


## manual

```shell

# add library
go get github.com/vigneshuvi/GoDateFormat
```
